elFinder_XH filebrowser Plugin for CMSimple

elFinder_XH is something that is Web-based file manager elFinder to plug-ins for CMSimple_XH.
elFinder operation and details of itself Please check with http://studio-42.github.io/elFinder/

Feature

  • elFinder 2.1.x *see demo
  • integrated editors:TinyMCE3/TinyMCE4/CKEditor
  • Include elFinder Plugin :AutoRotate/AutoResize/Watermark/Normalizer/Sanitizer
  • Include elFinder Theme：windows-10/moono/bootstrap
  • Custom style OK!

Requirements

elFinder 2.1.x

  • jQuery / jQuery UI
    jQuery 1.8.0+ jQuery UI 1.9.0+ However, we recommend newest version.
  • Client
    Modern browser. elFinder was tested in Firefox 12, Internet Explorer 8+, Safari 6, Opera 12 and Chrome 19
  • Server
    PHP 5.2+ (Recommend PHP 5.4 or higher) And for thumbnails - GD / Imagick module / convert(imagemagick) require
  • For Plugin
    Plugin:Normalizer PHP >= 5.3.0、 PECL intl >= 1.0.0 or Pear I18N_UnicodeNormalizer

Installation

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory to your server into the CMSimple_XH directory.
    • plugins/elfinder_xh/* ----> plugins/elfinder_xh/*.
  4. Test as a Default elFinder reference
    1. Change plugins/elfinder_xh/elfinder/php/connector.minimal.php-dist
      to
      plugins/elfinder_xh/elfinder/php/connector.minimal.php
    2. Change plugins/elfinder_xh/elfinder/elfinder.html_
      to
      plugins/elfinder_xh/elfinder/elfinder.html
    3. Check Your browser URL: http://your site･･･/plugins/elfinder_xh/elfinder/elfinder.html
  5. After completion of the test
    Delete(Or, change is not presumed name)
    plugins/elfinder_xh/elfinder/php/connector.minimal.php
    plugins/elfinder_xh/elfinder/elfinder.html

Settings

In Admin palnel "Plugins"Menu --> "Elfinder_xh".

  1. Interface

    • Width : The width of the elFinder main interface(in pixels or auto)
    • Height : The height of the elFinder main interface (in pixels).
    • Theme : theme(skin)
      • elfinder
        Dafult( recommend)
      • windows-10
        An elFinder theme that mimics Windows 10's user interface
        lokothodida/elfinder-theme-windows-10
      • moono
        A theme for elFinder that mimics the Moono CKEditor skin
        lokothodida/elfinder-theme-moono
      • bootstrap
        Bootstrap and LibreICONS theme
        StudioJunkyard/elfinder-boostrap-theme
    • ThumbnailSize
      Thumbnails size (in pixels). Thumbnails are square. It becomes necessary CSS and etc. fixes. because it is safer not to change with the exception of the special cases
    • ShowDotFilesDirs
      Show Files and Directories that start with a dot (.) it is safer not check with the exception of the special cases

  2. Trouble-Shoot

    • Xh-TemplateIsBootstrap
      If you are using Bootstrap type of template, Try Check on. Maybe ,Conflict between Bootstrap and jQuery UI .

    • Encoding
      This volume's local encoding. (server's file system encoding) It's necessary to be valid value to iconv.
      ex1. ISO-8859-1 , ex2. CP932 . Default empty(=UTF-8)

  3. Plugin-AutoRotate

    Auto rotation on file upload of JPEG file by EXIF Orientation..
    • Enable : Check to work

  4. Plugin-AutoResize

    Auto resize on image file upload.
    • Enable : Check to work
    • MaxWidth : MaxWidth .To resize . Greater than or equal to this size
    • MaxHeight : MaxHeight .To resize Greater than or equal to this size

  5. Plugin-Watermark

    Print watermark on image file upload.
    • Enable : Check to work
    • Source : Watermark PNG file name.
      Installation location : elfinder_xh/images/
      Ex: logo.png ( elfinder_xh/images/logo.png is used)
    • MarginRight : Watermark's Right position. Margin pixel.
    • MarginBottom : Watermark's Bottom position.Margin pixel.

  6. Plugin-Normalizer

    UTF-8 Normalizer of file-name and file-path etc.What?
    * This plugin require Class "Normalizer"(PHP 5 >= 5.3.0, PECL intl >= 1.0.0) or PEAR package "I18N_UnicodeNormalizer"
    • Enable : Check to work
    • Nfc : Normalization Form C：Canonical Decomposition followed by Canonical Composition
    • Nfkc : Normalization Form KC: Compatibility Decomposition followed by Canonical

  7. Plugin-Sanitizer

    Sanitizer of file-name and file-path etc.
    \ / ? " \< \> | change to _
    • Enable : Check to work
