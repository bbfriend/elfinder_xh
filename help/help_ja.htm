elFinder_XH filebrowser Plugin for CMSimple

elFinder_XH は、Webベースのファイルマネージャ elFinderを CMSimple_XH用プラグインにしたものです。
elFinderそのものの動作や詳細は http://studio-42.github.io/elFinder/ で確認してください。

特徴

  • elFinder 2.1.x *see demo
  • エディター対応：TinyMCE3/TinyMCE4/CKEditor
  • elFinderのプラグインを含む:AutoRotate/AutoResize/Watermark/Normalizer/Sanitizer
  • elFinderのテーマ含む：windows-10/moono/bootstrap
  • カスタム・スタイル

必要事項

elFinder 2.1.x

  • jQuery / jQuery UI
    jQuery 1.8.0+ jQuery UI 1.9.0+ However, we recommend newest version.
  • Client
    Modern browser. elFinder was tested in Firefox 12, Internet Explorer 8+, Safari 6, Opera 12 and Chrome 19
  • Server
    PHP 5.2以上 ( PHP 5.4以上を推奨) 、エクステンション GD または Imagick(imagemagick)が必要です
  • プラグイン
    プラグイン：Normalizerを使用するのであれば、PHP >= 5.3.0、 PECL intl >= 1.0.0 または Pear I18N_UnicodeNormalizer が必要です

Installation

The installation is done as with many other CMSimple_XH plugins. See the CMSimple_XH wiki for further details.

  1. Backup the data on your server.
  2. Unzip the distribution on your computer.
  3. Upload the whole directory to your server into the CMSimple_XH directory.
    • plugins/elfinder_xh/* ----> plugins/elfinder_xh/*.
  4. elFinder単体としてのテスト　参考
    1. plugins/elfinder_xh/elfinder/php/connector.minimal.php-dist を
      plugins/elfinder_xh/elfinder/php/connector.minimal.php に変更
    2. plugins/elfinder_xh/elfinder/elfinder.html_ を
      plugins/elfinder_xh/elfinder/elfinder.html に変更
    3. ブラウザを使用し URL： http://your site･･･/plugins/elfinder_xh/elfinder/elfinder.html でテスト
  5. テスト終了後､
    plugins/elfinder_xh/elfinder/php/connector.minimal.php
    plugins/elfinder_xh/elfinder/elfinder.html の２つを削除（または、推測されない名称に変更）

Settings

管理画面で　"Plugins"メニューより "Elfinder_xh".

  1. Interface
    • Width : 操作画面の 幅（単位:px または auto )
    • Height : 操作画面の 高さ (単位:px)
    • Theme : テーマ（スキン）
      • elfinder
        elfinder標準のテーマ（推奨）
      • windows-10
        An elFinder theme that mimics Windows 10's user interface
        lokothodida/elfinder-theme-windows-10
      • moono
        A theme for elFinder that mimics the Moono CKEditor skin
        lokothodida/elfinder-theme-moono
      • bootstrap
        Bootstrap and LibreICONS theme
        StudioJunkyard/elfinder-boostrap-theme
    • ThumbnailSize : サムネイルサイズ (単位:px)。サムネイルは正方形です。CSSの修正等が必要となりますので特別な場合を除いて変更しないほうが安全です
  2. Plugin-AutoRotate
    JPEG画像の自動回転。EXIF Orientationでアップロード時に処理します..
    • Enable : チェックして働きます
  3. Plugin-AutoResize
    画像の自動サイズ。アップロード時に処理します.
    • Enable : チェックして働きます
    • MaxWidth : 最大幅 .このサイズ以上の幅は、このサイズに変更
    • MaxHeight : 最大高さ .このサイズ以上の高さは、このサイズに変更
  4. Plugin-Watermark
    画像にウォーターマーク（すかし）を入れる
    • Enable : チェックして働きます
    • Source : ウォーターマーク（すかし）の画像名(png).
      設置場所 : elfinder_xh/images/
      例: logo.png ( elfinder_xh/images/logo.png が使用されます)
    • MarginRight : ウォーターマーク（すかし）の右の配置場所。Margin pixel.
    • MarginBottom : ウォーターマーク（すかし）の下からの配置場所.Margin pixel.
  5. Plugin-Normalizer
    ファイル名やPath名等のUTF-8 Normalizer
    * 動作させるには、 Class "Normalizer"(PHP 5 >= 5.3.0, PECL intl >= 1.0.0) または PEAR package "I18N_UnicodeNormalizer"が必須です
    • Enable : チェックして働きます
    • Nfc : 正規化形式C：文字は正準等価性によって分解され、再度合成。結果として文字の並びが変換前と変わることもありうる。
    • Nfkc : 正規化形式KC：文字は互換等価性によって分解され、正準等価性によって再度合成される。
  6. Plugin-Sanitizer:ファイル名やPath名等のサニタイズ。 / ? " \< \> | を _ に自動変更します
    • Enable : チェックして働きます

Update

elFinderに大きな変更が無い限り、elFinder本体のバージョンアップは可能です。

  1. ダウンロード：>=2.1.11 Releases · Studio-42/elFinder
  2. Unzip the distribution on your computer.
  3. Upload :elFinder-2.1.**/* -----> elfinder_xh/elfinder/*
